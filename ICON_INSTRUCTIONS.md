# Android Icon Sizes Required

## Tamaños exactos necesarios:

- `mipmap-mdpi/ic_launcher.png` → 48x48px
- `mipmap-hdpi/ic_launcher.png` → 72x72px  
- `mipmap-xhdpi/ic_launcher.png` → 96x96px
- `mipmap-xxhdpi/ic_launcher.png` → 144x144px
- `mipmap-xxxhdpi/ic_launcher.png` → 192x192px

## Mismo para íconos redondos:
- `mipmap-mdpi/ic_launcher_round.png` → 48x48px
- `mipmap-hdpi/ic_launcher_round.png` → 72x72px
- `mipmap-xhdpi/ic_launcher_round.png` → 96x96px
- `mipmap-xxhdpi/ic_launcher_round.png` → 144x144px
- `mipmap-xxxhdpi/ic_launcher_round.png` → 192x192px

## Herramientas recomendadas:
1. **Android Asset Studio**: https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html
2. **App Icon Generator**: https://appicon.co/
3. **Photoshop/GIMP**: Redimensionar manualmente

## Ubicación en el proyecto:
```
android/app/src/main/res/
├── mipmap-mdpi/
├── mipmap-hdpi/
├── mipmap-xhdpi/
├── mipmap-xxhdpi/
└── mipmap-xxxhdpi/
```
